# Firefly Mobile App Releases

Welcome to the **Firefly Mobile App Releases** repository!

This repository does not contain source code. It serves exclusively as a **secure hosting mechanism and CDN** for distributing compiled Android APK files for the Firefly Enterprise ecosystem.

## How to Publish a New APK

1. Go to the **Releases** tab on the right sidebar of this repository.
2. Click **Draft a new release**.
3. Create a new tag for the version (e.g., `v1.2.0-laugfs`).
4. Write a brief title and paste any relevant release notes in the description.
5. **Drag and drop your `.apk` file** into the "Attach binaries" section.
6. Click **Publish release**.

After publishing, right-click the `.apk` file in the Assets section, select "Copy Link Address", and paste that URL into the Firefly Admin Dashboard for the specific company!

## Security

Please do not upload sensitive source code, keystores, or internal configurations to this repository. Only upload the final, obfuscated `.apk` files.
